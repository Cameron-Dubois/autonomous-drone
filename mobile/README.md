# Mobile app (Expo / React Native)

Control and connect UI for the project drone. **BLE** (DroneBLE) and **Wi‑Fi** utilities live under `src/comms/`.

- **LiteWing (ESP32-S3) pairing and CRTP/UDP assumptions:** [docs/litewing-pairing.md](docs/litewing-pairing.md)

For a real device build (BLE, native Wi‑Fi), use a **development build** — see Expo / EAS setup in the repo.
